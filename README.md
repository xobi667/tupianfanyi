# 图片翻译器

这是一个用于电商图片翻译与重绘的 Next.js 项目仓库。

## 仓库结构

- `project/`
  真实可运行项目目录
- `启动图片翻译器.bat`
  根目录启动器，双击即可进入 `project` 并启动本地服务
- `codex/`
  开发过程、踩坑记录和变更日志

## 如何启动

直接双击根目录的：

- `启动图片翻译器.bat`

启动器会：

1. 进入 `project`
2. 检查 `node` / `npm`
3. 如有需要安装依赖
4. 启动本地开发服务
5. 自动打开浏览器

默认会打开：

- `http://localhost:3006`

## 项目代码位置

如果你要看源码或手动运行，请进入：

- `project`
