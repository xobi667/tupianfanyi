# 图片翻译器

这是一个基于 Next.js 的图片翻译与重绘工具，流程如下：

1. 自动识别图片中的原文
2. 将原文翻译成目标语言
3. 把原图交给 AI 生图模型，擦除原文并在原位置按接近原风格重绘译文

## 本地运行

前提：

- 已安装 Node.js

启动方式：

- 直接双击 `启动图片翻译器.bat`
- 或在项目目录执行 `npm install` 后再执行 `npm run dev`

浏览器地址：

- `http://localhost:3000`

## 配置说明

在页面右上角的设置里可以配置：

- API Key
- API Base URL
- API Key 传递方式
- 文本模型
- 生图模型

支持官方 Gemini 接口，也支持大多数 Gemini 兼容网关。

## 调试

开发模式下，服务端终端会输出更详细的请求日志，包括：

- 文本请求 / 生图请求标签
- 生图重试的兼容请求格式
- 上游状态码
- finish reason
- 是否返回图片
- 原始响应摘要
